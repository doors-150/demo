Before you start coding...

Download Visual Studio Code
After that, create a new file and make sure it ends with .html
Then you can start.

!DOCTYPE html

This should always be the very first thing you put in an html project.

It always set the HTML version for the website to the latest one.

This prevents any problems that might happen with older versions.



HTML, head, body

These are the three main parts of all of your code.

You put almost everything in HTML.

Things like the title go in head.

The main parts of your code go in body.



Header, Footer, Figure, Article

These are all semantic tags. They make your code easier to read and organize where everything is.

Header will be at the top. It is where things like the navbar and introduction will be.

Footer is the exact opposite, at the bottom. It is where you put copyright and how the user can contact you.

Figure is where an image will be.

And article is where you put texts, like sentences and paragraphs.
